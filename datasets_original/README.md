## Datasets

This folder contains three datasets used in our experiments.

The following table summarizes the datasets, including the number of attributes, protected attributes, and their sources:

| Dataset                     | #Attr. | Protected Attributes | Link                         |
|-----------------------------|--------|----------------------|------------------------------|
| Census Income (Adult)       | 13     | Age, Race, Gender    | https://doi.org/10.24432/C5XW20 |
| Bank Marketing (Bank)       | 16     | Age                  | https://doi.org/10.24432/C5K306 |
| German Credit Data (Credit) | 20     | Age, Gender          | https://doi.org/10.24432/C5NC77 |

The Adult dataset originally contains missing values. Instances with missing values have been removed.
