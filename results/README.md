# results

This directory stores the experimental results generated by the REDI framework, as well as scripts for post-processing and analysis.

Note: This directory contains only scripts and folder structures by default.  
Experimental results are generated automatically by running `REDI.py`.

## Structure

Results are organized as:
`results/<MODEL>/<DATASET>/<PROTECTED>/`


Each directory contains:

- `<MODEL>_<DATASET>_<PROTECTED>_<VALIDITY>.txt`  
  Raw results for each validity level

- `<MODEL>_<DATASET>_<PROTECTED>_<VALIDITY>_average.txt`  
  Mean and 95% confidence interval (generated by `average_results.py`)

- `<MODEL>_<DATASET>_<PROTECTED>_analysis.txt`  
  Correlation and regression analysis results (generated by `analysis.py`)

## Scripts

- `average_results.py`  
  Computes summary statistics (mean and confidence intervals)

- `analysis.py`  
  Performs correlation and regression analysis across validity levels
